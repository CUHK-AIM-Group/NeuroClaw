NeuroClaw Overview

Why NeuroClaw

Executable End to End

NeuroClaw turns neuroimaging work into reproducible research loops across data preparation, model execution, validation, and refinement.

Skills Guide the Agent

The skill library organizes tools, datasets, and model routes so the agent can choose the right action instead of guessing.

Audit-Friendly by Design

Every run leaves traces, reports, and intermediate artifacts, making results easier to inspect, compare, and debug.

Broad Coverage, Clear Entry Points

From ADNI and HCP to single-modality and multimodal pipelines, NeuroClaw keeps common workflows accessible in one place.

Skill Library Coverage

1. Basic Processing

1.1 Data and BIDS: DICOM↔NIfTI conversion, BIDS organization, metadata checks.

1.2 Environment and Engineering: Conda/dependency setup, Docker options, Git and Overleaf workflows.

1.3 Literature Search: academic retrieval and multi-engine evidence collection.

2. Tool-Centric Skills

2.1 FreeSurfer + FSL: reconstruction, segmentation, registration, GLM/ICA routes.

2.2 DIPY + HCP Pipelines: diffusion denoising, tensor/tract processing, multimodal handoff.

2.3 Integrated Stack: Nilearn, fMRIPrep, QSIPrep, XCP-D, CONN, MNE, MARS.

3. Modality-Centric Skills

3.1 sMRI: T1/T2 structural preprocessing, segmentation, cortical parcellation, WMH support.

3.2 fMRI: denoising, ROI extraction, connectivity analysis, GLM routes.

3.3 DWI + EEG: diffusion metrics/tractography plus EEG cleaning and feature extraction.

4. Dataset-Centric Skills

4.1 ADNI: acquisition, BIDS organization, preprocessing, ROI analysis, derived datasets.

4.2 HCP: structural/functional/diffusion execution with multimodal integration.

4.3 Expansion Targets: UKB, ADHD200, ABCD, ABIDE, COBRE, PNC, HBN, UCLA.

5. Model-Centric Skills

5.1 Deep Learning: BrainGNN, FM-APP, NeuroStorm for phenotype prediction.

5.2 Non-Deep-Learning: GLM, ICA, DictLearning, SVM, SpaceNet, clustering, filtering, detrending.

5.3 Routing: unified model entry with preprocessing dependency checks.

6. Research Workflow Skills

6.1 Single-Modality Pipelines: end-to-end sMRI/fMRI/DWI from raw data to outputs.

6.2 Dataset-Specific Pipelines: ADNI/HCP multimodal chains with QC checkpoints.

6.3 Research Output: method design, writing support, and auditable experiment logs.
