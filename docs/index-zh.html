覆盖技能库与任务体系

1. 基础处理

1.1 数据格式转换与 BIDS 标准化：BIDS 组织、DICOM↔NIfTI 转换、元数据提取。

1.2 环境配置与工程工具：Conda 环境管理、依赖解析、Git、Overleaf。

1.3 文献与信息检索：学术论文检索、多引擎搜索。

2. 工具相关

2.1 结构与功能工具链：FreeSurfer 与 FSL 覆盖皮层重建、分割、配准、GLM 与 ICA 等核心流程。

2.2 扩散与 HCP 集成：DIPY 与 HCP 流水线支持去噪、张量拟合、纤维建模、束追踪与多模态协同处理。

2.3 集成分析工具栈：Nilearn、fMRIPrep、QSIPrep、XCP-D、CONN、MNE、MARS 支持 ROI/连接分析、预处理、EEG 与统计流程。

3. 模态相关

3.1 sMRI：T1/T2 重建、组织分割、皮层分区、白质病灶分割、结构预处理。

3.2 fMRI：预处理、去噪、ROI 提取、静息态/任务态连接性、有效连接性、GLM 分析。

3.3 DWI 与 EEG：DWI 涵盖涡流校正、张量拟合、FA/MD、束追踪与连接组；EEG 包含频谱分析、伪迹去除与特征提取。

4. 数据集相关

4.1 ADNI：数据获取、BIDS 整理、预处理、ROI 提取、分析、VQA 数据集生成。

4.2 HCP：数据下载、BIDS 整理、结构/功能/扩散处理、多模态集成。

4.3 其他公共数据集：尚未覆盖 UKB、ADHD200、ABCD、ABIDE、COBRE、PNC、HBN、UCLA。

5. 模型相关

5.1 深度学习：BrainGNN、FM-APP、NeuroStorm 等模型路线，支持表型预测（分类/回归），输入可为 fMRI ROI 特征与可选 sMRI 特征。

5.2 非深度学习：GLM、ICA、DictLearning、SVM、SpaceNet、K-means、Hierarchical、Filtering、Detrending，用于激活分析、分解建模、分类、脑分区与信号去噪。

5.3 执行编排：统一模型入口负责预处理依赖匹配、输入输出映射与可执行计划生成，覆盖深度与传统方法。

6. 研究工作流相关

6.1 端到端单模态工作流：从原始数据到最终分析结果的完整 sMRI/fMRI/DWI 流水线。

6.2 端到端数据集特定工作流：完成 ADNI 或 HCP 多模态处理链并集成结果。

6.3 科研产出支持：文献调研、学术写作辅助、衍生数据集生成。
